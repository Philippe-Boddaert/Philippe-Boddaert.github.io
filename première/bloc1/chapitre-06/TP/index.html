    TP - Détective oulipien

    Contexte

    Le 24 Novembre 1960, des écrivains ont créé l'Oulipo (Ouvroir de littérature potentielle), un groupe d'écriture dans lequel ils expérimentèrent des contraintes d'écriture. Leur but était de découvrir de nouvelles potentialités du langage et de moderniser l’expression à travers des jeux d’écriture.

    Sont répertoriées ci-dessous certaines de ces contraintes, parmi les plus connues :

    • Anagramme : Une anagramme est un mot ou une expression obtenu en permutant les lettres d'un mot ou d'une expression de départ.
    • Anaphrase : Se dit d'une phrase contenant des anagrammes.
    • Lipogramme : Figure de style qui consiste à produire un texte d’où sont délibérément exclues certaines lettres de l’alphabet.
    • Monoconsonnantisme : Se dit d'un texte utilisant une seule consonne.
    • Pangramme : Se dit de phrases qui contiennent toutes les lettres de l’alphabet.
    • Palindrome : Texte qui peut se lire indifféremment de gauche à droite et de droite à gauche. Il existe des palindromes de syllabes, des palindromes de mots, de phrases ou de sons.
    • Surdéfinition : On définit doublement un mot : par son sens d’abord, mais aussi par sa présence à l’intérieur d’un autre mot.
    • Tautogramme : Texte dont tous les mots commencent par la même lettre.

    Consignes

    Vous êtes un détective Oulipien, amateur de littérature, dôté de capacité d'analyse et de programmation vous permettant de résoudre les énigmes qui vous sont proposées.

    Vous avez à disposition des textes dans le répertoire textes, mais le mystère plane sur les figures de style utilisées.

    Vous devez retrouver pour chaque texte, le ou les figures de styles utilisées :

    Vous allez créer un prédicat pour chaque figure de style et écrire dans la partie preuve de l'enquête, l'appel à la fonction qui permet de certifier (i.e le prédicat renvoie True) que le texte comporte la figure de style associée au prédicat.

    Aide

    Pour vous aider à traiter les pièces, le module enquete vous est fourni. Il permet d'ouvrir les fichiers textes et d'obtenir leurs contenus sous la forme d'une chaine de caractères.

    Des exemples d'utilisation :

    1
2
3
4
5
    >>> from enquete import Enquete
>>> Enquete.obtenir_piece(0)
'Ce chien colérique croque ces chaussures.'
>>> Enquete.obtenir_piece(8)
'Tata tatoue Toto : Titi. Tétue, Tata tatoue toutou Titi : Tutu.\n'

    Annexe - Rappel de fonctions utiles

    Pour rappel, quelques fonctions utiles à la manipulations des chaines de caractères en Python.

    replace

    • Signature : s.replace(old, new)
    • Spécifications : renvoie une copie de s avec toutes les occurrences de old remplacées par new.
    • Exemple : 
      1
2
      >>> "toutou".replace('u','i')
'toitoi'

    split

    • Signature : s.split(separator)
    • Spécifications : renvoie une liste de mots de s, utilisant separatorcomme délimiteur.
    • Exemple : 
      1
2
      >>> "toutou".split('u')
['to', 'to']

    lower

    • Signature : s.lower()
    • Spécifications : renvoie une copie de s, où chaque lettre est en minuscule.
    • Exemple : 
      1
2
      >>> "ToUTOu".lower()
'toutou'

    upper

    • Signature : s.upper()
    • Spécifications : renvoie une copie de s, où chaque lettre est en majuscule.
    • Exemple : 
      1
2
      >>> "ToUTOu".upper()
'TOUTOU'

    sorted

    • Signature : sorted(s)
    • Spécifications : renvoie une copie de s, dont les lettres sont triées par ordre alphabétique.
    • Exemple :
    1
2
    >>> "".join(sorted('monsieur'))
'eimnorsu'
