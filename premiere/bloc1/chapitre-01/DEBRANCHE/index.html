    Les systèmes de numération

    A travers l'histoire et les civilisations, l'humanité a eu la nécessité de compter dans divers domaines (scientifique, commerce, agriculture, ...), pour :

    • Faire des calculs scientifiques (Mathématique, Physique...)
    • Dater les événements (Histoire, ...)
    • Vie quotidienne : organiser les récoltes, échanger de la monnaie, ...

    Il exite une multitude de systèmes de numération, représentés dans la frise chronologique suivante :

    Cette activité débranchée vise à étudier les différents systèmes.

    Travail à faire

    Par groupe (4-5), vous devez étudier un système qui vous sera attribué et effectuez les tâches suivantes :

    • Lire la description du système de numération,
    • Représenter les entiers suivants selon votre système de numération : \(9, 42, 255\) et \(7659\)
    • Écrire l'algorithme de la fonction conversion qui prend un entier en paramètre et renvoie la représentation de l'entier dans la base de numération, sous la forme d'une suite de symboles.

    S1 - Numération additive

    Dans ce type de numération, on additionne les valeurs représentées par les symboles.

    Premier type

    On dispose de symboles pour désigner les puissances de la base \(b\) :

    $1\quad b\quad b^2\quad b^3 \quad b^4 \ldots $

    Les symboles sont groupés et ordonnés de gauche à droite.

    En base 10

    1. Numération Egyptienne

      • Symboles :

      • -3000 à -2900

      • Pas de zéro

    2. Numération Elamite (Iran)

      • Symboles :

      • -2900

      • Pas de zéro

    3. Numération Crêtoise

      • Symboles :

      • -1900 à -1350

      • Pas de zéro

    4. Numeration Hittite

      • Symboles :

      • -1400

      • Pas de zéro

    En base 20

    1. Numération Aztèque

      • Symboles :

      • 1200

      • Pas de zéro

    Second type

    On dispose d'une base \(b\) et de symboles représentants les nombres

    \(1 \quad k\quad b\quad k\times b\quad b^2\quad k\times b^2\quad b^3\quad k\times b^3\ldots\)

    où \(k\) est un diviseur de la base

    Base 10

    1. Grec acrophonique

      • \(k=5\)
      • Symboles :

      • -500
      • Pas de zéro

    2. Romane

      • \(k=5\)

      • Symboles :

      • -500

      • Pas de zéro

    Base 60

    1. Sumérienne

      • \(k=10\)

      • Symboles :

      • -3300

      • Pas de zéro

    Troisième type

    On dispose d'une base \(b\) et de symboles représentant les nombres :
    \(1\) \(1 \times b\) $ 1 \times b^2$ \(1 \times b^3\) \(\cdots\)
    \(2\) \(2 \times b\) $ 2 \times b^2$ \(2 \times b^3\) \(\cdots\)
    \(3\) \(3 \times b\) $ 3 \times b^2$ \(3 \times b^3\) \(\cdots\)
    \(\cdots\) \(\cdots\) \(\cdots\) \(\cdots\) \(\cdots\)
    \(b-1\) \((b-1) \times b\) $ (b-1) \times b^2$ \((b-1) \times b^3\) \(\cdots\)

    Base 10

    1. Egyptien hieratique

      • Symboles :

      • -2500

      • Pas de zéro

    2. Egyptien démotique

      • Symboles

      • -750

      • Pas de zéro

    3. Alphabétique grec

      • Symboles

      • -400

      • Pas de zéro

    4. Hébreu alphabétique

      • Symboles

      • -200

      • Pas de zéro

    5. Alphabétique arménien

      • Symboles

        - +400 - Pas de zéro

    S2 - Numération hybride

    Premier type

    Dans ce type de numération, la valeur est déterminée par le nombre d'unités placées à gauche des symboles représentant les puissances de la base \(b\).

    Base 10

    1. Assyro-Babilonien

      • symboles :

      • -2350

    2. Araméen

      • symboles :

      • -750

      • de droite à gauche.

    Deuxième type

    Dans ce type de numération, on dispose de 9 symboles pour désigner les unités, que l'on place à gauche des puissances de la base \(b\).

    Base 10

    1. Chinois commun

      • symboles :

      • -1400

      • pas de zéro

    S3 - Numération positionnelle

    Dans ce type de numération, c'est la position des symboles qui détermine la puissance de la base \(b\) par laquelle il faut multiplier la valeur du symbole.

    Premier type

    Dans ce type, on cumule encore les symboles des unités pour désigner les petites valeurs

    Base 10

    1. Chinois érudit

      • symboles :

      • -200

      • zéro : oui, tardif

    Base 20

    1. Maya érudit

      • symboles :

      • 400-900

      • Existence du zéro
      • De haut en bas.

    Second type

    Un symbole pour chaque valeur entre 1 et \(b-1\).

    Base 10

    • symboles :

    • 400

    • Existence du zéro
